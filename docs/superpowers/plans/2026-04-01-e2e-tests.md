**File:** `tests/test_e2e_cli.py`

### Fixture

```python
import subprocess, os, pytest

@pytest.fixture
def e2e(tmp_path):
    db = str(tmp_path / "e2e.db")
    env = {**os.environ, "TIMEOPT_DB": db}
    def run(*args):
        return subprocess.run(
            ["uv", "run", "timeopt", *args],
            capture_output=True, text=True, env=env,
            cwd="<project_root>",
        )
    return run

def seed(run, *raw_texts):
    """Seed tasks by calling the dump command with a mocked LLM — NOT needed,
    use direct DB seeding instead (open conn, core.dump_task)."""
```

Seeding note: open `db.get_connection(db_path)` + `core.dump_task()` directly — faster than LLM mock and valid for e2e since we're testing CLI output, not dump.
